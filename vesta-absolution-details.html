A 6 Month Gauntlet

This project was a huge endeavor with me being the lead programmer and having a hand in every single system we created. I learned about Git and version control which, hoo-boy, with students is a task and a half. The meat and potatoes for this game however was my, custom OpenGL shader programming, Animation Rigging and Animation State Management programming, Character Controllers, Level Design, 3D Character Modelling, and AI. Many of these systems I learned and improved on as we progressed. We also did weekly sprints following an Agile Methodology, adhering to the core principles.
